# Cucumber & Playwright

This test suite shows how to use Cucumber (Gherkin) with Playwright in Node.js.

Setup:
1. npm install
2. npx playwright install chromium

Run:
- npm test          (headless)
- npm run test:headed  (headed, with PWDEBUG=1 for debugging)
